# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-06-27

### Added
- Initial release of Orchis Rice
- Automated installer for Orchis Dark GTK Theme
- Automated installer for Tela Circle Dark Icons
- Automated installer for Bibata Modern Ice Cursor
- Automated installer for Inter and JetBrains Mono fonts
- Automated installation and configuration of GNOME Extensions (Blur My Shell, Dash to Dock, ArcMenu, Just Perfection, AppIndicator)
- GNOME desktop configuration applying themes, fonts, window behaviors, and dock settings
- Interactive wallpaper selection menu
- Clean uninstaller script to revert changes
- Continuous integration with ShellCheck
